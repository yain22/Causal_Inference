Percentile interval

[qα/2,m*,q1−α/2,m*]

The endpoints are empirical quantiles of the successful bootstrap estimates. The percentile interval respects strictly monotone transformations, but it is generally only first-order accurate and can miscover when estimator bias or skewness is substantial.

Normal / Wald interval

Δ^m±z1−α/2SE^boot,m

This interval is symmetric around the original estimate. With the same standard error and critical value, it is algebraically dual to the corresponding two-sided Wald test.

Interpretation of the bootstrap bias estimate: mean(Δ̂*) − Δ̂ estimates local finite-sample bias under the empirical resampling law. It is not an estimate of causal bias from unmeasured confounding, poor overlap, or persistent model misspecification.

The displayed formula is for the two-sided alternative; one-sided versions use the corresponding signed inequality. This unstudentized test is approximate and is not generally dual to the percentile interval.

CI/test duality

RejectH0⇔Δ0∉C1−α

This equivalence requires a matching dual pair. The two-sided Wald test matches the two-sided normal/Wald interval; directional Wald tests match the corresponding one-sided lower or upper confidence bound.

Independent resampling units

The unit bootstrap assumes subjects are i.i.d. If data are clustered, longitudinal, paired, or survey sampled, resample the appropriate cluster or use a design-respecting method.

Regular estimator

The ordinary bootstrap can fail for nonregular estimators, boundary parameters, extreme model selection, or some highly adaptive machine-learning procedures without additional theory.

Causal identification precedes inference

Bootstrap uncertainty does not repair unmeasured confounding, poor overlap, measurement error, nuisance-model bias, or a mismatch between the estimand and analysis.

Interpretation: a narrow bootstrap interval can be precisely centered on the wrong causal value. The simulation-only sampling-distribution panel makes that failure visible when hidden confounding or model misspecification is introduced.
Failed fits: summaries use only the Bs successful bootstrap analyses. Frequent failures are themselves a warning about instability or nonregularity; simply discarding many failed fits can distort the empirical bootstrap distribution.

Se Yoon Lee, Ph.D. • Synthetic teaching data. The displayed superpopulation ATE is a fixed-seed Monte Carlo approximation. The nonparametric bootstrap quantifies sampling uncertainty under its resampling and regularity assumptions; causal interpretation additionally requires consistency, exchangeability, positivity, and appropriate estimators. Unit-level resampling is not suitable for clustered, paired, longitudinal, or complex-survey data without modification.
